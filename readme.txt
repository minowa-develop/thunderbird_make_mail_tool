-- 初期設定 --
1."report.conf" に 氏名(NAME),サンダーバードのパス(THUNDERBIRD_PATH),宛先アドレス(TO) を入力
2."report_body.txt" にテンプレのメッセージを入力
  空白のみの行はDOSの仕様により詰められてしまうので 全角空白(　) か non break space( ) を入力する
  ※ 日付や氏名は置換によってデータの注入を行っている、注入キーワードを参照

-- 使用方法 --
1.make_report.bat を押下すると初期設定で入力した本文や氏名がサンダーバードの下書きとして表示される
2.メール本文を本日の内容に変更し、送信する

-- 注入ワード --
{NAME} 氏名
{YEAR} 現在の年
{MONTH} 現在の月
{YEAR} 現在の日

-- ファイル一覧 --
- make_report.bat
  実行ファイル、処理が記述している
- report.conf
  設定ファイル、初期設定として各項目に入力する必要がある
- report_body.txt
  メール本文のテンプレートファイル、データの注入により氏名や日付を入れ込んでいる
- readme.txt
  これです
